    获取竞猜赛事详细信息  ¶

    获取竞猜赛事详细信息
    POST/guessEventInfo

    返回数据格式(Array) ¶

    • eventDetail (Object) - 赛事详细

      • name (String) - 赛事名称
      • time (String) - 比赛用时
      • thumb (URL) - 赛事LOGO
      • matchName (String) - 比赛名称

    • games (Array) - 比赛列表

      • completed (boolean): 比赛是否已完成
      • gameTime (String): 比赛用时
      • group (String): 比赛分组
      • matches (String): 比赛场次
      • team (Array): 比赛队伍
        • currentScore (Number): 当前得分分数
        • currentScoreWin (Boolean): 当前得分是否胜利
        • first (Boolean): 是否发球
        • score (Number): 得分
        • win (Boolean): 是否胜利
        • winGameNumber (Number): 胜利场次
        • user (Array): 队员(双打传一组数据，双打两组)
          • username (String): 用户名
          • Userimage (String): 用户头像

    获取竞猜押注信息  ¶

    获取竞猜押注信息
    POST/bettingInfo

    接口有待商议 ¶

    返回数据格式(Array) ¶

    • status (Number) - 比赛状态 1:比赛即将开始 2:比赛已经开始 3:比赛已经结束

    • eventName (Strnig) - 赛事名称

    • matchName (String)- 比赛名称

    • time (String) - 比赛开始时间

    • location (String) - 比赛地址

    • gameTime (String) - 比赛用时

    • integral (Number) - 可用积分

    • match (String) - 比赛双方名字 例:陈韵竹 VS 李思

    • profit (Number) - 投注收益比例

    • risk (Boolean) - 风险 true 风险高显示为绿色 false风险低显示为红色

    • score (比分) - 押注比分 例: 对3:0进行押注
